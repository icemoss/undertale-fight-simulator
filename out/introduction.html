Undertale Fight Simulator

This Undertale Fight Simulator aims to be a faithful recreation of the battle system found within Undertale.

This project was created by the Godot engine, check it out!

Controls:

  • Menu Navigation - Arrow Keys
  • Accept Menu - Z
  • Exit Submenu - X
  • Move Soul - Arrow Keys
  • Move Soul Slowly - X
